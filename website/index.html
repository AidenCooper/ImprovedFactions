MySQL

Mysql allows the plugins to operate faster on large server. It also provides a way of syncing up with the database, so that the plugin can operate on a bungee-cord server network

High performance

This plugin got designed with performance in mind. It's 3x better in performance than other popular faction plugins - And all of at no cost of features and customizable

The plugin has an intelligent way for handling data. It stores quick access caches in RAM, while keeping the RAM heavy data stored and only loaded when needed - Just as someone would expect it
